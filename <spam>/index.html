<spam> tag

Purpose

You want to know how to do this in your own html?

First, you need javascript, so copy and paste this:

<script> document.querySelectorAll('spam').forEach(el => { const times = parseInt(el.getAttribute('times')) || 1; const separator = el.getAttribute('separator') || ''; const text = el.textContent; el.textContent = Array(times).fill(text).join(separator); }); </script>

Then, to actually use it as if it was an html element, use the

<spam></spam> tags with the parameter times to specify how much times the thing is repeated, for example <spam times="9">oo</spam>. There is also an optional parameter separator, which specifies which thing separates the repeating stuff, for example <spam times="9" separator=" ">oo</spam>, but the times parameter is not optional.
