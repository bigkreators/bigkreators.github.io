All IPA Symbols

The International Phonetic Alphabet (IPA) is a set of symbols that linguists use to describe the sounds of spoken languages.

This page lets you hear the sounds that the symbols represent, like ipachart.com, but with an addition of the a diacritics chart, which doesn't have audio, except the dark l and voiced bilabial approximant, and a modified chart that brings all the dedicated symbols into one place, but note that this doesn't include the diacritic combiniations nessacary for filling in the rest of the chart, except the affricates, lateral affricates, and ejectives. Still remember that it is only a rough guide.

Hit a symbol to hear the sound it represents.

For best results, please use a browser with HTML5 Audio support. The latest version of any popular web browser will do.
